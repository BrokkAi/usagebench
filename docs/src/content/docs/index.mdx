UsageBench currently focuses on a task that maps naturally to LSP operations:
declaration-to-reference lookup, reference-to-declaration navigation, and type
navigation. Call graphs, taint, and other analyses should use focused sibling
benchmarks rather than stretching “usage” into an analyzer-specific umbrella.
